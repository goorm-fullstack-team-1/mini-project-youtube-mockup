# Mini Project: YouTube UI Mockup

JS 강의를 통해 자바스크립트의 기본기를 학습한 후, 실제 유튜브 화면을 참고하여 **HTML과 CSS만으로 기본 UI 목업(Mockup)을 제작**하는 미니 프로젝트입니다.  
이번 프로젝트를 통해 브라우저가 HTML/CSS를 어떻게 해석하고 렌더링하는지 이해하며, UI/UX 디자인의 기초 역량을 다집니다.

---

## 🛠️ 습득 역량
- HTML 마크업 기술
- CSS 스타일링 기술
- UI/UX 디자인 기초
- 와이어프레임 기반 디자인
- 브라우저 렌더링 원리 파악

---

## 👥 협업 진행 방식
